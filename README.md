## Exercício `JurosSimples`

Calcule o montante final de um investimento a juros simples.

j = c . i . t

m = c + j

```c
Juros simples (j)

Capital [c] (R$).......: 1200,00
Taxa de juros [i] (%)..: 2
Tempo [t] (meses)......: 15

Juros (R$).....: 360,00
Montante (R$)..: 1560,00
```
